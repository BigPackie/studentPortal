ICIT Student portal mobile application

Version: 0.5

Application is not yet complete. Some features and content might be missing, bugs might be present. Please report all the bugs via email.

The purpose of this application is to inform students about news and ongoing promotions as well as their points received by using ICIT Services. Points may be exchanged for rewards by visting the reward exchange kiosk and displaying the chosen reward's code on the rewards tab.
